# NurePhysEduTest


Це інтерактивний вебдодаток для вимірювання різних типів реакції та моторики. У додатку доступні три види тестів: реакція на світло, реакція на звук та тепінг-тест. Вони дозволяють оцінити швидкість нервово-м’язової відповіді, частоту рухів та концентрацію. Дані автоматично зберігаються локально, а результати можна зручно порівнювати та експортувати.

Додаток побудований так, щоб тестування було простим, точним та цікавим — як для навчальних цілей, так і для самоспостереження та експериментів.
